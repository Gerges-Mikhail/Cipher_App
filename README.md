# Cipher_App
This application is a set of well-known ciphers that encrypt and decrypt the message you enter
Algorithms included in the application:
<br>1- Caesar.</br>
<br>2- Playfair.</br>
<br>3- Auto Key</br>
<br>4- Vigenere.</br>
